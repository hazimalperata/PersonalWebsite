## Overview

I developed the 1likte Seller Panel entirely from scratch, designing a scalable frontend architecture that supports dynamic product management, order tracking, and real-time UI updates. The panel was built with a focus on maintainable code, responsive layouts, and seamless user interactions.

## Key Features

- **Dynamic Product Management**: Added CRUD operations for products, inventory control, and image uploads.
- **Order Tracking**: Implemented real-time updates for order statuses and notifications.
- **Responsive UI**: Converted Figma designs into pixel-perfect, fully responsive pages.
- **Secure Authentication**: Managed role-based access control and secure login flows.

## Technologies Used

- **Next.js & React**
- **TypeScript**
- **Tailwind CSS**
- **Zustand / React Query**

## Challenges and Learnings

Ensuring scalability while maintaining clean, reusable components was a key challenge. Mapping complex Figma designs into interactive React components required attention to detail and testing across multiple devices.

## Outcome

The Seller Panel became the central hub for merchants on the platform, improving workflow efficiency and creating a consistent user experience.
