## Overview

The Admin Panel was built from zero to allow platform administrators to manage users, monitor analytics, and control operational workflows. I focused on performance, usability, and maintainable architecture while converting Figma designs to functional interfaces.

## Key Features

- **User Management**: Role-based access and administrative controls.
- **Analytics Dashboard**: Data visualization for operational insights.
- **Reusable Components**: Created a component library for future scalability.
- **Security & Performance**: Optimized for secure and fast operations.

## Technologies Used

- **Next.js & React**
- **TypeScript**
- **Tailwind CSS**
- **Server-Side Rendering**

## Challenges and Learnings

Implementing complex data tables and dashboards required balancing performance with UX. Ensuring secure access while keeping interfaces responsive was critical.

## Outcome

Admins gained a powerful, intuitive platform to manage the entire 1likte ecosystem efficiently.
