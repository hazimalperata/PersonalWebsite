## Overview

I built the 1likte mobile application using Expo and React Native, translating Figma designs into functional mobile interfaces. The app includes navigation, offline support, API integration, and push notifications, providing seamless access to platform features on mobile devices.

## Key Features

- **Cross-Platform Access**: Works on both iOS and Android.
- **Navigation Flows**: Implemented intuitive user flows and screens.
- **Offline Support**: Handles network interruptions gracefully.
- **Store Deployment**: Managed app store submission and deployment.

## Technologies Used

- **React Native & Expo SDK**
- **TypeScript**
- **Expo Router**
- **Push Notification APIs**

## Challenges and Learnings

Balancing performance, responsiveness, and code maintainability across platforms was a challenge. Handling offline scenarios and asynchronous API calls required robust state management.

## Outcome

The mobile app extended 1likte’s reach to mobile users, improving engagement and accessibility for the platform.
