## Overview

*Adventures Of The Knight* is a 2D platformer developed in Godot as a hands-on project to learn game development and take a first step into the game industry. The goal was to understand core gameplay systems while building something fun and playable.

The game follows a simple but engaging loop: run, jump, defeat enemies by jumping on their heads, and collect coins while progressing through handcrafted levels.

## Key Features

- **Classic Platforming Mechanics**: Smooth movement, jumping, and physics-based platform interactions.
- **Enemy System**: Inspired by classic platformers, enemies can be defeated by jumping on top of them.
- **Collectibles**: Coins placed throughout levels for exploration and scoring.
- **Level Design**: Multiple handcrafted levels with increasing difficulty.
- **Animations**: Character and environment animations for a more lively experience.
- **Audio**: Background music and sound effects to enhance gameplay feel.
- **Pause & Main Menu**: Fully functional UI flow including pause menu and main menu.
- **Cross-Platform Support**: Built for both mobile and desktop platforms.
- **Responsive Controls**: Optimized input handling for keyboard and touch devices.

## Technologies Used

- **Godot Engine**
- **GDScript**
- **2D Physics System**
- **UI System (Godot Control Nodes)**
- **Audio Engine (Godot Audio Server)**

## Challenges and Learnings

One of the main challenges was learning how to properly structure a game project in Godot, especially managing scenes, player state, and level transitions. Implementing responsive controls for both mobile and desktop also required careful input design.

Through this project, I gained practical experience in game loops, collision systems, animation handling, and basic game architecture.

## Outcome

This project served as a strong introduction to game development fundamentals. It helped solidify my understanding of Godot and gave me confidence to continue building more complex and polished games in the future.
